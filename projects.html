ByteView

Display any file, group of files, or folder as a bitmap image made from the raw bytes of the file. Save and load pictures, sort them and extract the unique colors. Export a series of images made from a group of files using the Large File Processor.

Latest release: Version 1.0.0 (released August 11, 2017)

IronArc

This project is Alpha and is still in development. IronArc is a virtual machine running a simple custom instruction set. The IronArc host runs multiple VMs and will provide access to the machines using the Debugger. Attach virtual and physical hardware (terminals, I/O, and disk access). This is not meant for production or serious use - it is a learning exercise in low-level programming.

Latest Update: Alpha v0.1 released!. Next: Working on a disassembler and debugger.

Cix

This project is pre-Alpha and is still in development. Cix (pronounced "six") is a compiler project for a simple C-like language. Its primary use will be to produce programs for the IronArc platform.

Latest Update: First stages of AST generation (structure definitons and function headers) are in progress.
